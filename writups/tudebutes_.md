Le code prend la chaine de caractères "flag_enc", le XOR avec 90 (0x5a) et la compare avec l'input de l'utilisateur.
Si la chaine de caractères est la même alors il renvoie un message de succès sinon erreur.
Pour obtenir le flag, nous avons simplement récupéré la valeur dans flag_enc et l'avons xoré (cyberchef) avec 90.
Le flag est : 
92I_34SYM0DD3BUT
